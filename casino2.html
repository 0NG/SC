空之轨迹SC赌牌心得

说明

原作者 ydslash 。还是提醒一下，小赌怡情，大赌伤身；利用这个研究研究概率论是个好事，沉迷其中就糟了= =

SC里面赌博赢钱最佳的方法就是赌牌，想靠轮盘和老虎机发家那是不可能地~原因在于赌牌可以通过连续的赢牌来获得筹码的不断放大，而其他的方式做不到这一点。

想要赢得99990点的唯一途径就是连续赢牌，而赢牌的基本方式可以分为3种：对子、顺子、同花。其他的方式只是上述3个方式的特例而已，不予特别讨论。下面的讨论以实际13*4=52张牌考虑。

第一轮5张牌赢牌的概率：
对子：1 - 52/52 * 48/51 * 44/50 * 40/49 * 36/48 = 0.4929 （这里面包括了3同和4同的概率）
顺子：9 * 20/52 * 16/51 * 12/50 * 8/49 * 4/48 = 0.0035
同花：4 * 13/52 * 12/51 * 11/50 * 10/49 * 9/48 = 0.0020
可以看出，想要赢牌，对子是主要的方式，其他的概率很低。

第二轮的情况：
如果第一轮5张牌里面没有赢牌的点，那么该怎么留牌？
先单纯考虑对子的情况：
在第一轮牌很乱的情况下（如1369k）
全不留：概率略小于0.4929
换一张：4*3/47=0.2553
换两张：1 - 38/47*37/46 + 32/47*3/46 + 6/47*2/46 = 0.3996
换三张：1- 41/47*40/46*39/45 + 32/47*(1-43/46*42/45+9/46*2/45+28/46*3/45) + 9/47*(1-44/46*43/45+6/46*2/45+32/46*3/45) = 0.4895
换四张：约0.49
可以看出：只换一张赢牌的概率最低为25%，换两张为40%，其他情况赢牌概率都接近50%。

如果考虑到顺子和同花，情况就稍有不同了
第一轮手里的牌没有对子，但有部分顺子或同花，如何处理？
留4张的情况：
凑成5张顺子的概率：8/47=0.1702（1234和KQJ10的概率低一半）
凑成5张同花的概率：9/47=0.1915
加上凑成对子的概率，赢牌的概率分别为：0.4255和0.4468
如果有3张同花或顺子并将其留下：
凑成5张顺子的概率：4/47*8/46+4/47*8/46+4/47*4/46+4/47*4/46 = 0.0444
凑成5张同花的概率：10/47*9/46=0.0416
所以赢牌的概率为：0.444和0.4412
留两张的情况：
凑成5张同花的概率：11/47*10/46*9/45=0.0102
顺子的概率应该差不多
所以总的赢牌概率为0.4998

考虑到顺子和同花的赔率较高，想要赢到99990（约折合人民币25000），两条原则：
1：有对留对（废话）。
2：无对留顺或同花（其实不用算也知道的）。
